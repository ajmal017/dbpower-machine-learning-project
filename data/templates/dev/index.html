Scope
To develop NQ LSTM machine learning tool that will be used to predict NQ(E-mini Nasdaq-100 futures) trend of coming 5 minutes.

Method
We will use 10,000 sets of 1-minute NQ data for original training, and then use the trained model to run 5000 sets 1-minute data, each set of data contains:

About 10 technical indicators, 50 Feature
For example, MACD/RSI/KDJ/PSAR/Bollinger/SMA/EMA/WMA, and each technical indicator may have more than 1 feature, for example, MACD will use T-9 to T to determine the strength of the trend, SMA/ EMA/WMA will use 20/50/100 averages, etc.

About 20 trend data and signals
For example, /Day High/Day Low/Day Open/Volume/Time/is Day High/is Day Low/above xx-SMA/Below xx-SMA
After the original training, we will adjust or delete the poorly performing Feature based on the training result The result of each group of training is T+5 – T> 0.5% ( Test Positive if the increase is greater than 0.5%)
